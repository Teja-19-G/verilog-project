# Verilog Projects

## 1. 4-bit Synchronous Counter

A synchronous 4-bit counter increments at clock every rising edge, from 0 to 15, and reset back to 0
### Features
- Synchronous reset
- Counts on rising clock edge
- 4-bit output vector

### Tools
- EDA Playground
- Icarus Verilog v12
